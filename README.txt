To modify the image file image.ex : 
1. Put image.ex in the same folder as imageglitching.py
2. Navigate to that folder in the terminal
3. Run imageglitching.py (ie python.exe imageglitching.py) 
4. Follow the prompts

Outputs modifications (3 panel images, ie 3 * width of the original) in the same folder. Can create multiple types of outputs at once.
